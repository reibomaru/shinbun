Color Usage Rules

OK - Color Allowed

  • Topic / Format バッジに定義済みカラーを使う
  • カテゴリ見出しのドットに Topic カラーを使う
  • 緊急アラートバナーに赤系を使う（唯一の例外）

NG - Gray Only

  • カード背景・ボーダーにカラーを付ける
  • アクションボタンのホバーにカラーを使う
  • 重要度スコアの星に黄色を使う
  • "Why it matters" に黄色背景を使う
  • 定義外の新しいカラーを独自に追加する

Exception: Urgent Alert Banner

緊急アラートバナーのみ、安全上の視認性確保のため赤系カラーを許容する。
ただし、緊急記事のカード自体はグレーボーダーのまま。タグ（Security / Incident）で識別する。

⚠ 緊急アラート — 重大なセキュリティ脆弱性が報告されました
